# SVG Bouncing Balls Animation

This project showcases a simple SVG animation of bouncing balls with interactive controls using HTML, CSS, and JavaScript. 

## Features

- **Multiple Bouncing Balls**: Animated balls with different gradient fills.
- **Responsive Design**: Scales with the viewport width.
- **Interactive Controls**: Buttons to start, stop, speed up, or slow down the animation.
- **Advanced Styling**: Includes gradients and shadows for enhanced visual appeal.

## Getting Started

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Anushri255/svg-bouncing-animation.git
    ```

2. **Open `svg.html` in your browser to view the animation.**
